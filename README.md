### WerwolfSwift
## Werwolf / TTT / Amongus Inspired Party game made with Swift.
![alt text](https://i.imgur.com/ybxGS68.jpg "Main Banner")

The whole App + all the graphics for the app were made only using an IPad Pro 2018.
Apps Used:
* Swift Playgrounds 5
* Adobe Photoshop
* Sceenshot
* Buttoncraft

## Used Packets:
